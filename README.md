Job Description:

We are seeking a highly skilled Senior React JS Developer to design and develop modern, scalable, and high-performance web applications. The ideal candidate will have strong expertise in React.js, modern JavaScript frameworks, UI architecture, and frontend best practices, along with experience collaborating in Agile development environments.

Key Responsibilities

 

Frontend Development

·        Develop responsive and high-performance web applications using React.js.

·        Build reusable components and scalable frontend architectures.

·        Translate UI/UX designs into clean, maintainable code.

·        Ensure cross-browser compatibility and mobile responsiveness.

Application Architecture

·        Design component-based architecture and state management strategies.

·        Optimize application performance and loading speed.

·        Implement secure and scalable frontend solutions.

Integration & APIs

·        Integrate frontend applications with RESTful APIs and backend services.

·        Work closely with backend developers to define API contracts.

·        Handle authentication, authorization, and data handling efficiently.

Code Quality & Best Practices

·        Write clean, testable, and maintainable code.

·        Perform code reviews and mentor junior developers.

·        Follow modern frontend development standards and design patterns.

Performance Optimization

·        Improve application performance using lazy loading, code splitting, and caching strategies.

·        Debug production issues and implement performance improvements.

Collaboration

·        Work within Agile/Scrum teams.

·        Collaborate with product managers, designers, and QA teams.

·        Participate in sprint planning, stand-ups, and retrospectives.

Required Skills & Experience

·        10+ years of frontend development experience.

·        Strong hands-on experience with React.js and modern JavaScript (ES6+).

·        Expertise in HTML5, CSS3, and responsive design principles.

·        Experience with state management tools (Redux, Context API, or similar).

·        Strong knowledge of REST APIs and asynchronous programming.

·        Experience with Git-based version control.

·        Understanding of web performance optimization techniques.

Preferred Skills

·        Experience with TypeScript.

·        Knowledge of Next.js or other React frameworks.

·        Experience with Node.js or full-stack development concepts.

·        Familiarity with cloud platforms (AWS, Azure, GCP).

·        Experience with testing frameworks (Jest, React Testing Library, Cypress).

·        CI/CD pipeline exposure.
